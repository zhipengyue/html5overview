null 与 undefined的 区别

null 为未赋值 undefined 为未定义 typeOf(null) 输出 'object' typeOf(undefind) 输出 'undefined' 同时对一个为定义的变量，进行typeOf操作,得到的同样是 'undefined'字符串 Number(null) 输出 0 Number(undefined) 输出 Nan typeOf(Nan) 输出 'number'

==与===

==如果类型不同，==会对数据进行类型转换，转换完进行===比较，如果相同则返回true否则返回false ===则先判断类型是否相同，类型不同直接返回false null==undefined //true
